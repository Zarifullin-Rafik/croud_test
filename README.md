# Анализ поиска в Яндекс Картинках

## 📊 О проекте
Анализ семпла запросов к Яндекс Картинкам для сравнения поведения пользователей на desktop и мобильных устройствах.

## 📋 Задачи
1. Определить диапазон дат в семпле
2. Подсчитать запросы с "ютуб" на разных платформах
3. Найти топ-10 запросов для каждой платформы
4. Проанализировать трафик по часам
5. Выделить контрастные тематики

## 🛠 Технологии
- Python (pandas, matplotlib, plotly, pandasql)
- Jupyter Notebook
- Yandex DataLens

## 📈 Основные выводы
- На мобильных устройствах чаще ищут поздравления и развлечения
- На компьютерах - образовательный контент и контект для решения рабочих задач
- Пик мобильного трафика - утро и вечер
- Пик десктопного трафика - день
